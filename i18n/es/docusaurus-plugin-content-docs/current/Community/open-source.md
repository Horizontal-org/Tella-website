# Tella es de código abierto

La seguridad y privacidad de los usuarios de Tella son fundamentales en nuestra misión. Por ello, es natural que el código de la aplicación sea completamente de código abierto. Esto significa que está disponible públicamente para su revisión: cualquiera puede auditar nuestro código para verificar que no hacemos un mal uso de los datos de los usuarios, que realmente hacemos lo que decimos y que nuestro código es robusto.

El código de Tella tanto para Android como para iOS se puede acceder [aquí en nuestro repositorio de GitHub](https://github.com/horizontal-org). En lugar de confiar en nosotros, puedes auditar el código tú mismo o pedirle a alguien de confianza que lo haga, o leer los [informes](/security-and-privacy#security-audits) que las empresas de seguridad independientes han escrito después de auditar el código de Tella.
