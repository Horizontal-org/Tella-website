# Tella Docs

Tella စာရွက်စာတမ်းများမှ ကြိုဆိုပါသည်။

အောက်တွင် Tella အလုပ်လုပ်ပုံ၊ ၎င်း၏ လုံခြုံရေးအင်္ဂါရပ်များ၊ ကန့်သတ်ချက်များနှင့် စတင်နည်းတို့ကို နားလည်ရန် လိုအပ်သည့် အရာအားလုံးကို သင်တွေ့ရပါမည်။ နောက်ဆုံးရ သတင်းများ သိရှိစေရန် [ကျွန်ုပ်တို့၏ သတင်းလွှာ](https://blog.wearehorizontal.org/#/portal) တွင် စာရင်းသွင်းပါ၊ [ကျွန်ုပ်တို့ထံ ဆက်သွယ်ပါ](/contact-us)။
